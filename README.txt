Login básico | Diseño de interfaz (UI)

Este proyecto es un diseño simple y limpio de una página de registro de usuarios, desarrollado utilizando HTML y CSS, con iconos provistos por el paquete Boxicons. El objetivo principal de este diseño es ofrecer una interfaz visualmente atractiva para un formulario de registro básico.

El formulario incluye:
- Un campo para ingresar el nombre de usuario.
- Un campo para ingresar la contraseña.
- Iconos visuales que acompañan los campos para mejorar la experiencia del usuario.
- Una opción para "Recordarme" y un enlace para recuperar la contraseña en caso de olvido.
- Un botón para crear una cuenta.
- Un enlace para iniciar sesión si ya tienes una cuenta.

Es importante destacar que este proyecto es únicamente un diseño visual, sin funcionalidades de backend ni manejo de datos.

Estructura del proyecto:
- index.html: Archivo principal que contiene la estructura del formulario de registro.
- style.css: Archivo de estilos que define la apariencia y el estilo de la interfaz.
- Boxicons: Se utiliza para agregar iconos visuales en los campos de entrada del formulario.

Instalación:
No se requiere ninguna instalación especial. Simplemente descarga o clona el repositorio, y abre el archivo "index.html" en tu navegador para ver el diseño.
